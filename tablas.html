Tablas

Con los TAGS TABLE, TR, TH y TD se crean tablas que organizan informacion para resumirla.

  • TR: Permite declarar renglones (rows)
  • TH: Declarar cabeceras de columnas
  • TD: Para declarar los datos de las celdas


Id Nombre Edad

Los atributos: border, caption, colspan y rowspan

  • BORDER: Establece el ancho de borde para la tabla
  • CAPTION: Establece un titulo arriba de la cabecera
  • COLSPAN: Permite que una celda ocupe varias columnas
  • ROWSPAN: Permite que una celda ocupe varios renglones
